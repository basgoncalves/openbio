================================================================================
                     WHAT WAS THE PROBLEM?
================================================================================

Local Buildozer Failures (WSL2):
  • SSL/TLS certificate verification errors
  • hostpython3 lacks SSL module (by design)
  • pyjnius couldn't download build dependencies
  • Workarounds (HTTP, PYTHONHTTPSVERIFY=0) ineffective
  • Root cause: Architectural limitation, not configuration

Why It Was Unfixable:
  • hostpython3 is intentionally minimal (for Android size constraints)
  • Lacks SSL support by design
  • Cannot be patched or worked around
  • Different from system Python (which has SSL)
  • Affects entire p4a native compilation pipeline

Why Cloud Build Solves It:
  • Cloud servers have properly configured Python
  • Complete SSL/TLS support
  • All build tools pre-installed (SDK, NDK, JDK)
  • Professional infrastructure (no environment issues)
  • 10x faster than local builds

🚀 BUILD THE APK
═══════════════════════════════════════════════════════════════════

Open WSL Ubuntu terminal and run:

    cd ~/poserecorder_build
    bash BUILD_NOW.sh

That's it! The script handles everything.

Time: 15-30 minutes
Don't interrupt: Build may appear stuck (wait 10+ minutes)

═══════════════════════════════════════════════════════════════════

📊 WHAT HAPPENS
═══════════════════════════════════════════════════════════════════

The script will:
  1. Install system dependencies (2-3 min)
  2. Setup buildozer (2-3 min)
  3. Clean old cache (1 min)
  4. Verify configuration (1 min)
  5. Build APK (12-20 min) ← LONGEST
  6. Report success or error

❌ IF BUILD FAILS
═══════════════════════════════════════════════════════════════════

Check the error message. Common fixes:

SSL Error:
  export PIP_INDEX_URL=http://pypi.org/simple/
  bash BUILD_NOW.sh

Space Error:
  rm -rf ~/.buildozer/
  bash BUILD_NOW.sh

Other Error:
  Read: DO_THIS_NOW.md → "If Build Fails"

═══════════════════════════════════════════════════════════════════

📱 INSTALL ON DEVICE
═══════════════════════════════════════════════════════════════════

After successful build:

1. Connect Android device (USB cable)
2. Enable developer mode on device
3. Run:
   adb devices              # Verify device listed
   adb install bin/poserecorder-0.2-debug.apk

4. On device:
   • Open "Pose Recording & Analysis"
   • Grant camera permissions
   • Tap "● START RECORDING"
   • Do a movement
   • Tap "■ STOP"
   • Wait for analysis
